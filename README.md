# Proyecto SIMANI

El objetivo principal  de este reporitorio es construir una aplicación utilizando React JS para la creación de gráficas utilizando Apache Echarts, adémas de solicitar la información al backend para su graficación.
También cuenta con la opción de crear PDF's y EXCEL's desde el cliente.

En este proyecto se encuentra package.json con las librerías como los archivos.js en la carpeta src.
